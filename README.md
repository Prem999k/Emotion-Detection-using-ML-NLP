# 💬 Emotion Detection using NLP & Machine Learning

## 📌 Project Overview
This project is an **NLP-based Emotion Detection System** that analyzes user input text and predicts the underlying emotion in real-time.

The system uses **Machine Learning (Logistic Regression)** with **TF-IDF feature extraction** and is deployed using **Streamlit** for an interactive web interface.



## 🌐 Live Demo

Access the deployed application here:

👉 https://emotion-detection-using-ml-nlp-premkumark.streamlit.app/
---

## 🎯 Features

- 🔹 Real-time emotion prediction
- 🔹 Clean and modern Streamlit UI
- 🔹 Text preprocessing using NLP techniques
- 🔹 TF-IDF vectorization
- 🔹 Machine Learning model (Logistic Regression)
- 🔹 Emoji-based output for better UX
- 🔹 Confidence score display
- 🔹 Fully deployed web application

## ⚙️ Tech Stack

- **Python**
- **NLP (NLTK)**
- **Scikit-learn**
- **TF-IDF Vectorizer**

- **Streamlit (Deployment UI)**
- **Joblib (Model Saving)**

---

## 🔄 Workflow

1. Data Collection
2. Data Cleaning & Preprocessing
   - Lowercasing
   - Removing punctuation & numbers
   - Stopwords removal
   - Lemmatization
3. Feature Extraction (TF-IDF)
4. Label Encoding (using LabelEncoder)
5. Model Training (Logistic Regression)
6. Model Evaluation
7. Deployment using Streamlit
